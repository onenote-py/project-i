Basic structure of the c program



First C Program

// Hello world program in c
#include <stdio.h>

int main()
{
  printf("Hello, World\n");
  return 0;
}

Basic structure of C program

  1. #include<stdio.h>
    2.

    #include is the pre-processor directive that is used to include files in our program. Here we are including the stdio.h standard file which is necessary for the declarations of basic standard input/output library in C programming.

  2. int main()

    3. The execution of any C program starts with the main function, hence it is necessary to have a main function in your program. ‘int’ is the return value of this function.

  3. {}

    4. The curly brackets are used to indicate the starting and ending point of any function. Every opening bracket should have a corresponding closing bracket.

  4. printf ("Hello, world\n");

    5. This is a C statement. printf represents the standard output stream in C. It is declared in the stdio.h The text between quotations will be printed on the screen. \n will not be printed, it is used to add line break. Each statement in C ends with a semicolon (;).

  5. return 0;
    6.

    return signifies the end of a function. Here the function is main, so when we hit return 0, it exits the program. We are returning 0 because we mentioned the return type of main function as integer (int main). A zero indicates that everything went fine and a one indicates that something has gone wrong.


Input and Output in C

The header file stdio.h must be included to make use of the input/output (printf/scanf) function.

  • Standard Output (printf)
    •

    By default, the standard output of a program points at the screen. So with the printf function can print a message onto the screen.

    Example:

     // standard output program
#include <stdio.h>

int main()
{
  printf("an example to print output on the screen");
  return 0;
}
  • Standard input (scanf)
    •

    In most of the cases the standard input device is the keyboard. With the function (scanf) it is possible to read input from the keyboard.

    Example:

    // standard input program
#include <stdio.h>

int main()
{
  char ch;
  printf("press a character and press return: ");
  scanf("%c",&ch);
  printf("%c",ch);
  return 0;
}
Note:
  • %c format specifier inside the scanf() function to take character input from the user.
  • When you press a character, it is stored in the ch variable.
  • The input is processed by scanf after the return key is pressed.

The scanf function will always return the variable type that you use with scanf. So if you request an integer you will get an integer and so on. also you can take multiple inputs from the user and display them as shown as below.

#include<stdio.h>
 
int main{
  int a;
  float b;
  printf("Enter integer and then a float: ");
  // Taking multiple inputs
  scanf("%d%f", &a, &b);
  printf("You entered %d and %f", a, b);
  return 0;
 }

program output

 Enter  integer and then a float: -3
 3.4
 You entered -3 and 3.400000

In this case the user must give two input values, that are separated by any valid blank separator (tab, space or new-line).
