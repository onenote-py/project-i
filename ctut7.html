Decision Making In C


Decision making is about deciding the order of execution of statements based on certain conditions or repeat a group of statements until certain specified conditions are met.

C language handles decision-making by supporting the following statements,

  • if statement
  • switch statement
  • conditional operator statement (? : operator)
  • goto statement

if statement

The if statement may be implemented in different forms depending on the complexity of conditions to be tested.
The different forms are,

  1. Simple if statement
  2. if....else statement
  3. Nested if....else statement
  4. Using else if statement


Simple if statement

Block of statement executed only when a specified condition is met.

The syntax of the if statement looks like:

 if(condition){
 // statement to be executed
  }

Example 0.1

//simple c program comparison between a and b
#include<stdio.h>

int main(){
    int a,b;
    a = 8, b = 4;
    if(a>b){
        printf("a is greater than b");
    }
    return 0;
}  

 Output:
 a is greater than b

If the expression returns true, then the if statement will get executed, otherwise if statement is skipped and only the statement-outside of the if statement will get executed.



if....else statement

When we want to execute some block of code if a condition is met and another block of code if a condition does not met, In such a case we use if….else statement.

Syntax:
   if(expression)
   {
      statement block1;
   }
   else
   {
      statement block2;
   }

Example 0.2


 //check number is even or odd
#include<stdio.h>

int main(){
    int num;
    printf("Enter a number : ");
    scanf("%d",&num);
    if(num%2==0){
        printf("%d is even",num);
    }
    else{
      printf("%d is odd",num);
    }
    return 0;
} 

output:
 Enter a number : 45
 45 is odd

If the expression is met, the statement-block1 is executed, else statement-block1 is skipped and statement-block2 is executed.



Nested if....else statement

When there is an if statement inside another if statement then it is known as nested if else. 

 Syntax:
  
if(expression)
{
  Statement 1;
}
else
{
  if(expression)
  {
    Statement 2;
  }
  else
  {
    Statement 3;
  }
}

Example

//program to compare the two number
#include<stdio.h>

int main(){
  int a, b;
  printf("Enter the value of a\n");
  scanf("%d",&a);
  printf("Enter the value of b\n");
  scanf("%d",&b);
 
  // outer if statement
  if ( a != b ) {
    // inner if statement
       if ( a > b ) {
         printf("a is greater than b") ;
          }
    // inner else statement
       else {
        printf ("b is greater than a");
         }
 } 
 //outer else statement
 else {
    printf("both values are equal");
 }
 return 0 ;
}
 output:
 Enter the value of a
 55
 Enter the value of b
 55
 both values are equal

Here the nested if else statement are used to specify conditions within conditions.

Note:

    As you can see, nested if...else makes your logic complicated. If possible, you should always try to avoid nested if...else.



Using else if statement

The if….elseif…else statement is used to select one of among several blocks of code to be executed.

Syntax:
if(expression1)
{
Statement 1;
}
else if(expression2)
{
Statement 2;
}
.
.
else
{
Statement n;
}

The expression is tested from the top(of the ladder) downwards. As soon as a true condition is found, the statement associated with it is executed.


Example

}
