Operators In C Programming



• Operators In C

Operators are symbols that perform operations on variables and values.

Operators are used for producing output by performing various types of calculations on two or more inputs.

  1. Arithmetic Operators
  2. Assignment Operators
  3. Relational Operators
  4. Logical Operators
C Arithmetic Operators

Arithmetic operators are used for performing mathematical operations on variables and data. like + , - , * ...etc.

Operator    Operation
  +     Addition
  -     Substraction
  *     Multiplication
  /      Division
  %     Modulo Operation (Remainder after division)

Example:

#include <stdio.h>

int main()
{
  int a = 9, b = 2;
  
  //the sum of a and b.
  printf("the value of a + b is %d\n",a+b);
  //the difference of a and b.
  printf("the value of a - b is %d\n",a-b);
  //the product of a and b.
  printf("the value of a * b is %d\n",a*b);
  //the division of a by b
  printf("the value of a / b is %d\n",a/b);
  //the modulo of a by b
  printf("the value of a % b is %d\n",a%b);
  return 0;
}

Program Output

 the value of a + b is 11
 the value of a - b is 7
 the value of a * b is 18
 the value of a / b is 4
 the value of a % b is 1
 /  Division Operator

Note the operation (a / b) in our program. The / operator is the division operator.

As we can see from the above example, if an integer is divided by another integer, we will get the quotient. However, if either divisor or dividend is a floating-point number, we will get the result in decimals.

Some Examples of division operation with float,

  9/2 is 3
  9.0 / 2 is 4.5
  9 / 2.0 is 4.5
  9.0 / 2.0 is 4.5
 %  Modulo Operator

The modulo operator % computed the remainder. When a = 9 is divided by b = 2 , the remainder is 1.

Note
  • The Modulo % operator can only be used with integers.


C Assignment Operators

assignment operators are used to assign values to variables

for example:  
    int a = 34;  // a is assigned 34.
Operator   Example     Equivalent to
  =   a = b;   a = b;  
  +=   a += b;   a = a + b; 
  -=   a -= b;   a = a - b; 
  *=   a *= b   a = a * b; 
  /=   a /= b;   a = a / b; 
  %=   a %= b;   a = a % b; 

Example

#include <stdio.h>

int main() {
    int a, b;
    a = 2;    // a is assigned 2
    b = 7;    // b is assigned 6
    printf("a = %d\n",a);
    printf("b = %d\n",b);
    // assigning the sum of a and b to a
    a += b;  // a = a + b
    printf("a = %d\n",a);

    return 0;
}

Program Output

 a = 2
 b = 7
 a = 9

C Relational Operators

A relational operator is used to check the relationship between two operands.

 example:
  // checks if a is greater than b
  a > b;

Here,  >  is a relational operator. It checks if a is greater than b or not.

If the relation is true, it returns 1 whereas if the relation is false, it returns 0.

  Operator     Meaning
 ==    Equals operator
 !=    Not Equals operator
 <     smaller than
 >     Greater Than
 <=    Less Than or Equal To
 >=     Greater Than or Equal To

Example

#include <stdio.h>

int main() {
    int a, b;
    a = 3;
    b = 5;
    printf("3 > 5 is %d\n",a>b);
    printf("3 < 5 is %d\n",a<b);
    printf("3 == 5 is %d\n",a==b);
    printf("3 != 5 is %d\n",a!=b);
    printf("3 >= 5 is %d\n",a>=b);
    printf("3 <= 5 is %d\n",a<=b);

    return 0;
}

Program Output

 3 > 5 is 0
 3 < 5 is 1
 3 == 5 is 0
 3 != 5 is 1
 3 >= 5 is 0
 3 <= 5 is 1
Note:

Relational operators are used in decision making and loops.



C Logical Operators

Logical operators are used for comparing two expressions. For example ((a==b) && (a>b)).

Also logical operators are used to check whether an expression is true or false. If the expression is true, it returns 1 whereas if the expression is false, it returns 0.

Operator    Meaning
  &&  Logical AND. True only if all the operands are true.
  ||  Logical OR. True if at least one of the operands is true.
  !  Logical NOT. True only if the operand is false.

Example

 // Working of logical operators

#include <stdio.h>
int main()
{
    int a = 5, b = 5, c = 10, result;

    result = (a == b) && (c > b);
    printf("(a == b) && (c > b) is %d \n", result);

    result = (a == b) && (c < b);
    printf("(a == b) && (c < b) is %d \n", result);

    result = (a == b) || (c < b);
    printf("(a == b) || (c < b) is %d \n", result);

    result = (a != b) || (c < b);
    printf("(a != b) || (c < b) is %d \n", result);

    result = !(a != b);
    printf("!(a != b) is %d \n", result);

    result = !(a == b);
    printf("!(a == b) is %d \n", result);

    return 0;
}

Program Output

 (a == b) && (c > b) is 1 
 (a == b) && (c < b) is 0 
 (a == b) || (c < b) is 1 
 (a != b) || (c < b) is 0 
 !(a != b) is 1 
 !(a == b) is 0 .

Explanation of logical operator program
  • (a == b) && (c > 5) evaluates to 1 because both operands (a == b) and (c > b) is 1 (true).
  • (a == b) && (c < b) evaluates to 0 because operand (c < b) is 0 (false).
  • (a == b) || (c < b) evaluates to 1 because (a = b) is 1 (true).
  • (a != b) || (c < b) evaluates to 0 because both operand (a != b) and (c < b) are 0 (false).
  • !(a != b) evaluates to 1 because operand (a != b) is 0 (false). Hence, !(a != b) is 1 (true).
  • !(a == b) evaluates to 0 because (a == b) is 1 (true). Hence, !(a == b) is 0 (false).
Note:

Logical operators are commonly used in decision making in C programming.



Other operators in c

The sizeof Operator

The sizeof is a unary operator that returns the size of data (constants, variables, array, structure, etc).

Example

 #include <stdio.h>
int main()
{
    int a;
    float b;
    double c;
    char d;
    printf("Size of int=%lu bytes\n",sizeof(a));
    printf("Size of float=%lu bytes\n",sizeof(b));
    printf("Size of double=%lu bytes\n",sizeof(c));
    printf("Size of char=%lu byte\n",sizeof(d));

    return 0;
}

Program Output

 Size of int = 4 bytes
 Size of float = 4 bytes
 Size of double = 8 bytes
 Size of char = 1 byte
