Variables, Constants and Keywords


Variables:

    A variable is a name given to the memory location (storage area) to hold data. Each variable should be given a unique name (identifier).

    For Example

    int a = 2; // a is assigned 2.

    Here a is the variable name that holds the integer value 2. The value of a can be changed, hence the name variable.

There are some certain rules for naming a variable in C:
  1. Can only have alphabets, numbers and underscore.
  2. Cannot begin with a number, No special symbol Allowed Other than underscore ( _ )
  3. Cannot begin with an uppercase character.
  4. Cannot be a keyword defined in C language (like int is a keyword).

Note
  • A C identifier is a name used to identify a variable, function, or any other user-defined items.
  • C does not allow punctuation characters such as @, $, and % within identifiers.
  • C is a case sensitive programming language. Thus, Swap and swap are two different identifiers in C.

Constants

    Mathematically constants is an entity whose value doesn’t change.

    Similarly we have an constants in c programming that value doesn't change.

Types of constant

    Primarily there are 3 types of constant:

  1. Integer Constant: An Integer constant can only be (whole numbers) without any decimal point ex:- ( -33, 56, 23, 124 )

  2. Real Constant: A real constant is an floating point number, only have decimal values ex:- ( -234.45, 2.4, 3.33 )

  3. Character Constant: Character Constant only have single character enclosed within single inverted commas ex:- ( '4’,’$’,’c’, '@')

Note:

    Character must be enclosed within single inverted commas


Keywords

Keywords are predefined, reserved words used in programming that have special meanings to the compiler and Keywords are part of the syntax.

For example:

  int age;  

Here, int is a keyword that indicates age is a variable of type int (integer).


C Keywords
auto double int struct
break else long switch
case enum register typedef
char extern return union
continue for signed void
do if static while
default goto sizeof volatile
const float short unsigned

Note:
  • Keywords cannot be used as an identifier.
  • all keywords must be written in lowercase.

Comments

Comments are like helping text in your C program and they are ignored by the compiler.

there are two types of comments, one is the single line comments and second is multiple line comments, single line comments are denoted by two forward slash // and multi lines comments start with /* and terminates with the characters */ as shown below:


  1. Single line comment
    2.  syntax:
       // Comments goes here
  2. Multi line comment
    3.  syntax:
       /* Comments goes here */
Note:
  • It is important that you choose a style of commenting and use it consistently throughout your source code. Doing so makes the code more readable.
  • You cannot have comments within comments and they do not occur within a string or character literals.
