Data Types and Type Modifiers


Data-Types In C:

Data types are declarations for variables. This determines the type and size of data associated with variables which is essential to know since different data types occupy different size of memory.

Primarily there are three types of data-types in C:
  1. Primitive Data-types
  2. Derived Data-types
  3. User Defined Data-types
1) Primitive Data-types

These data types are built-in or predefined data types and can be used directly by the user to declare variables.

Example: int, char, float etc...

Primitive data types available in C are:

1) int
  • This data type is used to store integers.
  • It occupies 4 bytes (32 bits) in memory.
  • It can store values from -2147483648 to 2147483647.
    Eg. int age = 18
2) float & double
  • Used to store floating-point numbers (decimals and exponentials)
  • The size of float (single precision float data type) is 4 bytes ( 32 bits ).
  • the size of double (double precision float data type) is 8 bytes ( 64 bits ).
  • Float is used to store upto 6 decimal digits whereas double is used to store upto 15 decimal digits.
    Eg. float pi = 3.14;
    double distance = 24E8 // 24 x 108;
3) char
  • This data type is used to store characters. It occupies 1 byte in memory.
  • Characters in C are enclosed inside single quotes'',
  • ASCII code is used to store characters in memory.
    eg : char ch = 'a';
Data Types Meaning Size (in bytes)
int Integer 4 bytes
float Floating-point 4 bytes
double Double Floating-point 8 bytes
char Character 1 bytes
void Empty 0 bytes

Derived Data Types

These are the data types that are derived from fundamental (or built-in) data types. For example, arrays, pointers, structure.

User-Defined Data Types

These are the data types that are defined by user itself. For example, typedef, union, enumeration, etc.

C Type Modifiers

Type modifiers are used to modify the fundamental data types.

Data Types Size (in bytes) Meaning
signed int 4 used for integers (equivalent to int)
unsigned int 4 can only store positive integers
short 2 used for small integers (range -32768 to 32767)
long int at least 4 used for large integers (equivalent to long int)
long double 8 used for large floating-point numbers
signed char 1 used for characters (guaranteed range -127 to 127)
unsigned char 1 used for characters (range 0 to 255)
