C Multidimensional Arrays

In C programming, you can create an array of arrays. These arrays are known as multidimensional arrays. For example,

float x[3][4];

Here,  x  is a two-dimensional (2d) array. The array can hold 12 elements. You can think the array as a table with 3 rows and each row has 4 columns.


Similarly, you can declare a three-dimensional (3d) array. For example,

float x[2][4][3];

Here, the array y can hold 24 elements.


Initialization of a 2d array
// Different ways to initialize two-dimensional array

int x[2][3] = {{1, 3, 0}, {-1, 5, 9}};
         
int x[][3] = {{1, 3, 0}, {-1, 5, 9}};
                
int x[2][3] = {1, 3, 0, -1, 5, 9};

Initialization of a 3d array

initialization of three-dimensional array in a similar way like a two-dimensional array. Here's an example,

int test[2][3][4] = {
    {{3, 4, 2, 3}, {0, -3, 9, 11}, {23, 12, 23, 2}},
    {{13, 4, 56, 3}, {5, 9, 3, 5}, {3, 1, 4, 9}}};


Example 1:Two-dimensional array to store and print values 
// C program to store temperature of two cities of a week and display it.
#include <stdio.h>
const int CITY = 2;
const int WEEK = 7;
int main()
{
  int temperature[CITY][WEEK];
printf("taking values\n\n");
  // Using nested loop to store values in a 2d array
  for (int i = 0; i < CITY; ++i)
  {
    for (int j = 0; j < WEEK; ++j)
    {
      printf("City %d, Day %d: ", i + 1, j + 1);
      scanf("%d", &temperature[i][j]);
    }
  }
  printf("\nDisplaying values: \n\n");

  // Using nested loop to display vlues of a 2d array
  for (int i = 0; i < CITY; ++i)
  {
    for (int j = 0; j < WEEK; ++j)
    {
      printf("City %d, Day %d = %d°c\n", i + 1, j + 1, temperature[i][j]);
    }
  }
  return 0;
}

Example 2:Three-dimensional array 
// C Program to store and print 12 values entered by the user

#include <stdio.h>
int main()
{
  int num[2][3][2];

  printf("Enter 12 values: \n");

  for (int i = 0; i < 2; ++i)
  {
    for (int j = 0; j < 3; ++j)
    {
      for (int k = 0; k < 2; ++k)
      {
        scanf("%d", &num[i][j][k]);
      }
    }
  }

  // Printing values with proper C Programming Short Notes/index.

  printf("\nDisplaying values:\n");
  for (int i = 0; i < 2; ++i)
  {
    for (int j = 0; j < 3; ++j)
    {
      for (int k = 0; k < 2; ++k)
      {
        printf("index[%d][%d][%d] = %d\n", i, j, k, num[i][j][k]);
      }
    }
  }

  return 0;
}

Note:
  1. An array holds elements that have the same data type.
  2. Array elements are stored in subsequent memory locations.
  3. Two-dimensional array elements are stored row by row in subsequent memory locations.
  4. Array name represents the address of the starting element.
  5. Array size should be mentioned in the declaration. Array size must be a constant expression and not a variable.
