Format Specifiers In C

The format specifiers are used in C for input and output purposes. Using this concept the compiler can understand that what type of data is in a variable during taking input using the scanf() function and printing using printf() function.

Here is a list of format specifiers.

Format Specifier Type
 %c  Character
 %s  Strings
 %d  Singed integer
 %i  Unsigned integer
 %f  Float values
 %u  Unsigned int
 %e or %E  Scientific notation of floats
 %g or %G  Similar as %e or %E
 %hi  Signed integer (short)
 %hu  Unsigned integer (short)

Format Specifier Type
 %lf  Double
 %Lf  Long double
 %lu  Unsigned int or unsigned long
 %lli or %lld  Long long
 %llu  Unsigned long long
 %o  Octal representation
 %p  Pointer
 %x or %X  Hexadecimal representation
 %n  Prints nothing
 %%  Prints % character
 %l or %ld or %li  Long


Note:

    We can add some other parts with the format specifiers.
    These are like below −

  1. A minus symbol (-) sign tells left alignment
  2. A number after % specifies the minimum field width. If string is less than the width, it will be filled with spaces
  3. A period (.) is used to separate field width and precision
